Const useAxios

  • useAxios(config?: Config | null): [useAxiosRequests, requestData, axiosError, isLoading]
    • Defined in useAxios.ts:14

    Custom Hook

    Takes in one optional parameter. If no parameter is used then the default configuration for axios requests will be used.

    Parameters

    • Default value config: Config | null = null

      Axios request configuration. See Axios

    Returns [useAxiosRequests, requestData, axiosError, isLoading]

    • Returns array of values. Index 0 is the request object used to make the 4 different axios requests. Index 1 is the value of the request. Index two is the error message if there was an error and a blank string if there wasn't one. Index 3 is a boolean indicating if the request is currently being made.
